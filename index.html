Origins

  • Started as a hobby project
  • Cobbled together freely available components
  • Crop management (seedings, transplantings, inputs, etc)
  • General record keeping

Big Picture Goals

Manage everything

  • Plantings
  • Animals
  • Equipment
  • Soil
  • (Bees, Maple, Mushrooms, ...)

Organize information

  • All in one place
  • Relate records to one another
  • Approach from multiple angles
  • Save your brain space!

Farm mapping

  • Draw fields, greenhouses, buildings, etc
  • Import shape files
  • Custom aerial imagery
  • Mapping of activities

Decision support

  • Make it easy to remember the past
  • Provide tools to plan for the future
  • Adapt to the unexpected
  • Provide recommendations

Data ownership

  • Data is valuable
  • You are in control
  • Share if you want to

Platform to build upon

  • Farmers
  • Researchers
  • Service providers

Global community

  • Free and open source
  • Language translations
  • Unit conversions
  • Anyone can contribute!

Free software definition

  1. The freedom to run the program as you wish, for any purpose (freedom 0).
  2. The freedom to study how the program works, and change it so it does your computing as you wish (freedom 1). Access to the source code is a precondition for this.
  3. The freedom to redistribute copies so you can help your neighbor (freedom 2).
  4. The freedom to distribute copies of your modified versions to others (freedom 3). By doing this you can give the whole community a chance to benefit from your changes. Access to the source code is a precondition for this.
